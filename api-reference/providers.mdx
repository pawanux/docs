Manage connected providers.

## List providers

```
GET https://www.tryunleashx.com/v2/provider/all
Headers:
  Authorization: Bearer <api_key>
```

### Try it (curl)

```bash
curl -X GET 'https://www.tryunleashx.com/v2/provider/all' \
  -H 'Authorization: Bearer <api_key>'
```
