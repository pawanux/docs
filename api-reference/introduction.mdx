Use and leverage UnleashX Voice AI using APIs through HTTP requests from any language in your applications and workflows.

UnleashX APIs feature consistent, resource‑oriented URLs, accept application/json request bodies, return JSON responses, and use standard HTTP response codes, authentication, and HTTP verbs.

## Authentication

- Login to the dashboard at `https://platform.unleashx.ai`
- Navigate to the Developers tab
- Click `Generate a new API Key`
- Save your API Key securely (it is shown only once)

### Using the API key

Include your API key as a Bearer token in the Authorization header:

```
Authorization: Bearer <api_key>
```

### Example authenticated request

```
GET https://api.unleashx.ai/agent/all
Headers:
  Authorization: Bearer <api_key>
```

## Pagination

Standard cursor‑based pagination is supported on list endpoints unless otherwise specified in the endpoint documentation.
