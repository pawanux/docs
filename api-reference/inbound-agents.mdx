title: Inbound Agents

Configure inbound agent handling.

## List inbound agents

```
GET https://api.tryunleashx.com/v2/inbound-agent/all
Headers:
  Authorization: Bearer <api_key>
```

### Try it (curl)

```bash
curl -X GET 'https://api.tryunleashx.com/v2/inbound-agent/all' \
  -H 'Authorization: Bearer <api_key>'
```
