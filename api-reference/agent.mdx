---
title: Agent APIs
---

Manage agents in UnleashX.

## List agents

```
GET https://api.tryunleashx.com/v2/agent/all
Headers:
  Authorization: Bearer <api_key>
```

### Try it (curl)

```bash
curl -X GET 'https://api.tryunleashx.com/v2/agent/all' \
  -H 'Authorization: Bearer <api_key>'
```

## Create agent

```
POST https://api.tryunleashx.com/v2/agent
Headers:
  Authorization: Bearer <api_key>
  Content-Type: application/json
Body:
  {
    "name": "Sales Assistant",
    "language": "en"
  }
```

### Try it (curl)

```bash
curl -X POST 'https://api.tryunleashx.com/v2/agent' \
  -H 'Authorization: Bearer <api_key>' \
  -H 'Content-Type: application/json' \
  -d '{
    "name": "Sales Assistant",
    "language": "en"
  }'
```
