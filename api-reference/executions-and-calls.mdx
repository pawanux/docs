---
title: Executions & calls data
---

Fetch call executions and analytics.

## List calls

```
GET https://api.tryunleashx.com/v2/call/all
Headers:
  Authorization: Bearer <api_key>
```

## Get a call by ID

```
GET https://api.tryunleashx.com/v2/call/{call_id}
Headers:
  Authorization: Bearer <api_key>
```
