---
title: Executions & calls data
---

Fetch call executions and analytics.

## List calls

```
GET https://www.tryunleashx.com/v2/call/all
Headers:
  Authorization: Bearer <api_key>
```

### Try it (curl)

```bash
curl -X GET 'https://www.tryunleashx.com/v2/call/all' \
  -H 'Authorization: Bearer <api_key>'
```

## Get a call by ID

```
GET https://www.tryunleashx.com/v2/call/{call_id}
Headers:
  Authorization: Bearer <api_key>
```

### Try it (curl)

```bash
curl -X GET 'https://www.tryunleashx.com/v2/call/{call_id}' \
  -H 'Authorization: Bearer <api_key>'
```
