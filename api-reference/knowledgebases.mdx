title: Knowledgebases
---

Manage knowledge sources for grounding.

## List knowledgebases

```
GET https://api.tryunleashx.com/v2/knowledgebase/all
Headers:
  Authorization: Bearer <api_key>
```

### Try it (curl)

```bash
curl -X GET 'https://api.tryunleashx.com/v2/knowledgebase/all' \
  -H 'Authorization: Bearer <api_key>'
```
