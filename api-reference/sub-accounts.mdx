---
title: Sub accounts (Enterprise)
---

Manage sub accounts for enterprise setups.

## List sub accounts

```
GET https://www.tryunleashx.com/v2/sub-account/all
Headers:
  Authorization: Bearer <api_key>
```

### Try it (curl)

```bash
curl -X GET 'https://www.tryunleashx.com/v2/sub-account/all' \
  -H 'Authorization: Bearer <api_key>'
```
