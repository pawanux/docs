> **In one line:** Connect Google Drive so your UnleashX agents can search, read, and create files.

| | |
|---|---|
| **Authentication** | OAuth |
| **Setup time** | ~2 minutes |
| **Difficulty** | Easy |
| **Best for** | Teams who store documents in Drive and want agents to find or reference them |

## 1. Overview

**What is Google Drive?**
Google Drive is Google's cloud storage for documents, spreadsheets, and files.

**What can you do once it's connected?**
Search for files, read document contents, and create new files or copies.

**Why connect it to UnleashX?**
Your agent can pull up a contract, summarize a document, or save call notes as a file — without you switching apps.

## 2. What you'll need

Before you begin, make sure you have:

- A Google or Google Workspace account.
- For Workspace accounts: an admin who allows third-party apps, if your org restricts them.

<Warning>
  **On a managed Workspace account?** Some organizations block third-party apps by default. If authorization fails, ask your Google Workspace admin to allow UnleashX.
</Warning>

## 3. Get your credentials

Google Drive uses **OAuth**, so there's nothing to copy. You'll approve access in the next section by signing in to Google directly.

<Info>
  You'll see Google's standard consent screen listing exactly what UnleashX can do. Review it before approving.
</Info>

## 4. Connect on UnleashX

<Steps>
  <Step title="Open Integrations">
    In UnleashX, go to **Integrations** and search for **Google Drive**.
  </Step>
  <Step title="Authorize">
    Click **Connect**, then **Authorize**.
  </Step>
  <Step title="Sign in to Google">
    Choose your account, review the requested permissions, and click **Allow**.
  </Step>
  <Step title="Assign to an agent">
    You'll return to UnleashX automatically. Assign Google Drive to the agent(s) that should use it.
  </Step>
</Steps>

✅ **You're connected** when the status shows a green **Connected** badge. The tools below are now available.

### Permissions / scopes requested

| Permission | Why UnleashX needs it |
|------------|----------------------|
| `drive.readonly` | To search and read your files |
| `drive.file` | To create files and edit ones the agent makes |

## 5. Available tools

| Tool | What it does | Changes data? |
|------|--------------|:---:|
| `search_files` | Find files by name or content | No |
| `read_file_content` | Read the contents of a file | No |
| `get_file_metadata` | Get a file's details (owner, type, modified) | No |
| `create_file` | Create a new file | ✏️ Yes |
| `copy_file` | Make a copy of a file | ✏️ Yes |

> ✏️ = creates or edits data

## 6. Example usage

**Example 1 — Find a document**

> You type: _"Find the Acme renewal contract in Drive."_

What happens: UnleashX uses `search_files` to locate it and returns the file.

**Example 2 — Save notes**

> You type: _"Save today's call notes as a new doc called 'Acme — June call'."_

What happens: UnleashX uses `create_file` to create it. You'll see a confirmation when it's done.

## 7. Permissions & data access

When you connect Google Drive, UnleashX can:

- Search and read your files.
- Create files and edit files it created.

It **cannot**:

- Permanently delete your files.
- Access other Google services (Gmail, Calendar) unless you connect them separately.

You can disconnect at any time from **Integrations → Google Drive → Disconnect**. Disconnecting immediately revokes UnleashX's access.

## 8. Troubleshooting

| Problem | What it means | How to fix it |
|---------|---------------|---------------|
| `401 Unauthorized` | The connection expired or was revoked | Reconnect from **Integrations → Google Drive** |
| `403 access_denied` | Your Workspace blocks the app | Ask your Google admin to allow UnleashX |
| File not found | The file is in an account or shared drive you didn't connect | Confirm the right account is connected and you have access |

Still stuck? See [Authentication basics](/mcp-server/authentication) or contact [cs@unleashx.ai](mailto:cs@unleashx.ai).

## 9. Frequently asked questions

**Are my files stored by UnleashX?**
UnleashX reads files only when a tool call needs them during a conversation; it does not copy your Drive.

**Can the agent edit any of my files?**
It can edit files it created and ones you explicitly open to it; it does not get blanket edit access to your whole Drive.

**Can multiple team members connect their own Drive?**
Yes. Each person authorizes separately, so access reflects that person's Google account.
