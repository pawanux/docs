The [Model Context Protocol (MCP)](https://modelcontextprotocol.io) is an open standard that lets your UnleashX voice agents use external tools — Slack, GitHub, your CRM, your calendar — as actions they can take during a conversation. Each integration below walks you through getting your credentials and connecting. No coding required.

<Note>
  New here? Start with [How connecting works](#how-connecting-works) below, then read [Authentication basics](/mcp-server/authentication) before picking a server.
</Note>

## How connecting works

<Steps>
  <Step title="Pick a server">
    Choose a tool from the catalog below.
  </Step>
  <Step title="Get your credentials">
    Generate an API key or sign in with OAuth on the tool's side.
  </Step>
  <Step title="Connect it to UnleashX">
    Paste the credentials into UnleashX and assign the server to an agent — done.
  </Step>
</Steps>

Once connected, the server's tools become available to any agent you attach it to. When a conversation needs an action, the agent calls the matching tool and uses the result in its reply.

<Info>
  MCP tools complement [custom function calls](/voice-agent-function-calls/custom-function-calls). Use MCP when a tool already exposes the actions you need; use custom function calls for one-off, agent-specific logic.
</Info>
